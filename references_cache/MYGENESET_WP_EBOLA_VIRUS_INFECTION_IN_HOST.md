# MYGENESET:WP_EBOLA_VIRUS_INFECTION_IN_HOST  WP_EBOLA_VIRUS_INFECTION_IN_HOST

**MYGENESET:WP_EBOLA_VIRUS_INFECTION_IN_HOST** — WP_EBOLA_VIRUS_INFECTION_IN_HOST (msigdb, C2:CP:WIKIPATHWAYS)

## Description

WikiPathways Ebola virus infection in host gene set (MSigDB C2:CP:WIKIPATHWAYS). This is a host-response disease pathway gene set covering the major molecular events during Ebola virus disease (EVD). The 129 member genes span five mechanistic arms: (1) innate immune sensing and type I interferon induction — RIG-I (DDX58), MDA5 (IFIH1), MAVS, TBK1, IRF3, IRF7; (2) JAK-STAT-mediated type I IFN signalling and interferon-stimulated gene (ISG) expression — STAT1, STAT2, IRF9, JAK1, TYK2, IFNAR1, IFNAR2, MX1, OAS family, IFIT family, ISG15; (3) viral entry into host cells — NPC1 (intracellular EBOV receptor), HAVCR1/ TIM1 (attachment factor), cathepsins B and L (CTSB, CTSL) for glycoprotein processing; (4) Ebola virus immune evasion via antagonism of IFN production — host signalling components hijacked by EBOV VP35 (TBK1, IRF3 axis) and VP24 (STAT1/karyopherin axis); and (5) inflammatory and cytokine responses — TNF, IL6, IL1B, and NFkB signalling driving the cytokine storm characteristic of severe EVD. As an infection pathway gene set it captures both host antiviral defence and viral immune antagonism, making it a rich resource for studying EBOV pathogenesis and host-directed therapeutic targets.
