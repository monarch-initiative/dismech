# MYGENESET:WP_FAMILIAL_HYPERLIPIDEMIA_TYPE_1  WP_FAMILIAL_HYPERLIPIDEMIA_TYPE_1

**MYGENESET:WP_FAMILIAL_HYPERLIPIDEMIA_TYPE_1** — WP_FAMILIAL_HYPERLIPIDEMIA_TYPE_1 (msigdb, C2:CP:WIKIPATHWAYS)

## Description

WikiPathways Familial Hyperlipidemia Type 1 gene set (MSigDB C2:CP:WIKIPATHWAYS, 18 genes). This pathway gene set represents the molecular machinery required for triglyceride-rich lipoprotein (chylomicron) hydrolysis at the capillary endothelium. The five causal proteins — lipoprotein lipase (LPL), its cofactors apolipoprotein C-II (APOC2) and apolipoprotein A-V (APOA5), the endothelial LPL-anchoring glycoprotein GPIHBP1, and the LPL maturation factor LMF1 — constitute the rate-limiting chylomicron-triglyceride hydrolysis apparatus. Loss-of-function variants in any of these five genes impair LPL enzymatic activity, causing the accumulation of chylomicrons in plasma and severe hypertriglyceridemia, the hallmarks of familial chylomicronemia syndrome (hyperlipoproteinemia type I). Additional set members encode apolipoproteins and structural components that participate in chylomicron metabolism and downstream lipid transport.
