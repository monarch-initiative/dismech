# MYGENESET:REACTOME_COMPLEMENT_CASCADE  REACTOME_COMPLEMENT_CASCADE

**MYGENESET:REACTOME_COMPLEMENT_CASCADE** — REACTOME_COMPLEMENT_CASCADE (msigdb, C2:CP:REACTOME)

## Description

Reactome complement cascade pathway gene set (MSigDB C2 curated pathway set). This is a pathway signature, not a cell-type signature. Member genes encode the classical pathway initiators (C1Q, C1R, C1S, C4, C2), the alternative pathway components (C3, CFB, CFD, CFP), the lectin pathway (MBL2, MASP1, MASP2), the terminal lytic sequence (C5–C9) forming the membrane attack complex, and key regulatory proteins (CFH, CFI, CD55, CD59, C1-INH). The pathway represents the full complement activation cascade of innate immunity: recognition, opsonization, anaphylatoxin generation, and target lysis.
