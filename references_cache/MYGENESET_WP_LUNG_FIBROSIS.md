# MYGENESET:WP_LUNG_FIBROSIS  WP_LUNG_FIBROSIS

**MYGENESET:WP_LUNG_FIBROSIS** — WP_LUNG_FIBROSIS (msigdb, C2:CP:WIKIPATHWAYS)

## Description

WikiPathways lung fibrosis gene set (MSigDB C2:CP:WIKIPATHWAYS, 63 genes). This disease pathway gene set represents the molecular framework of pulmonary fibrosis, capturing the core signalling and effector arms: TGF-β-driven fibroblast-to-myofibroblast activation via the canonical SMAD cascade, excessive extracellular matrix and collagen deposition causing progressive tissue scarring, repetitive alveolar epithelial injury with dysregulated repair, and the downstream loss of pulmonary function. Member genes include TGF-β receptors (TGFBR1, TGFBR2), SMAD transcription factors, structural collagens, fibronectin, matrix metalloproteinases, and their regulators, reflecting the canonical profibrotic cascade in idiopathic pulmonary fibrosis and related interstitial lung diseases.
