# MYGENESET:WP_22Q112_COPY_NUMBER_VARIATION_SYNDROME  WP_22Q112_COPY_NUMBER_VARIATION_SYNDROME

**MYGENESET:WP_22Q112_COPY_NUMBER_VARIATION_SYNDROME** — WP_22Q112_COPY_NUMBER_VARIATION_SYNDROME (msigdb, C2:CP:WIKIPATHWAYS)

## Description

WikiPathways gene set representing the 22q11.2 copy number variation syndrome pathway (MSigDB C2:CP:WIKIPATHWAYS, 130 genes). This is a disease pathway gene set, not a cell-type expression signature. The member genes reflect the multi-system biology of 22q11.2 deletion syndrome (DiGeorge/velocardiofacial syndrome), caused by hemizygous deletion of chromosome 22q11.2. The primary pathological driver is haploinsufficiency of TBX1, a T-box transcription factor required for pharyngeal apparatus development. TBX1 loss disrupts neural crest cell migration and the organogenesis of the cardiac outflow tract, thymus, parathyroid glands, and craniofacial structures. The gene set additionally encodes downstream transcriptional regulators, signal transduction effectors, and neurodevelopmental factors including DGCR8 (a microRNA biogenesis component linked to schizophrenia risk), collectively capturing the cardiac, immunological, craniofacial, and neurodevelopmental dimensions of this syndrome.
