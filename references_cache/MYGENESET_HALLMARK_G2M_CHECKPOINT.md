# MYGENESET:HALLMARK_G2M_CHECKPOINT  HALLMARK_G2M_CHECKPOINT

**MYGENESET:HALLMARK_G2M_CHECKPOINT** — HALLMARK_G2M_CHECKPOINT (msigdb, H)

## Description

MSigDB Hallmark gene set of genes involved in the G2/M cell-cycle checkpoint and mitosis. This is a PROCESS signature, not a cell-type signature; the member genes encode core regulators of the G2-to-M phase transition (CDK1, CCNB1/B2, CDC25A/B/C, WEE1, PKMYT1), spindle assembly checkpoint proteins (BUB1, BUB1B/BUBR1, MAD1L1, MAD2L1), mitotic kinases (AURKA, AURKB, PLK1), and chromosome condensation and segregation factors. The hallmark represents the proliferative cell-cycle programme active in all dividing human cell types.
