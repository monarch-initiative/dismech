# MYGENESET:WP_ALZHEIMERS_DISEASE  WP_ALZHEIMERS_DISEASE

**MYGENESET:WP_ALZHEIMERS_DISEASE** — WP_ALZHEIMERS_DISEASE (msigdb, C2:CP:WIKIPATHWAYS)

## Description

WikiPathways Alzheimer's disease pathway gene set (MSigDB C2 curated gene set; WikiPathways WP5124). This is a disease pathway gene set, not a cell-type signature. Spanning 261 genes, it is substantially broader than the KEGG legacy Alzheimer's disease set and represents all major mechanistic arms of AD pathophysiology: amyloid precursor protein processing (APP, BACE1, PSEN1, PSEN2, ADAM10), tau hyperphosphorylation and neurofibrillary tangle formation (MAPT, CDK5, GSK3B), neuroinflammation (IL1B, TNF, TREM2), synaptic dysfunction (GRIN1, GRIN2A, GRIN2B, CHRNA7), apoptosis (CASP3, CASP9, BAX), mitochondrial and oxidative stress, and calcium signaling.
