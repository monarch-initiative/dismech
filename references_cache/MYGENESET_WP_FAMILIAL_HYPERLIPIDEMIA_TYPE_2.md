# MYGENESET:WP_FAMILIAL_HYPERLIPIDEMIA_TYPE_2  WP_FAMILIAL_HYPERLIPIDEMIA_TYPE_2

**MYGENESET:WP_FAMILIAL_HYPERLIPIDEMIA_TYPE_2** — WP_FAMILIAL_HYPERLIPIDEMIA_TYPE_2 (msigdb, C2:CP:WIKIPATHWAYS)

## Description

WikiPathways Familial Hyperlipidemia Type 2 gene set (MSigDB C2:CP:WIKIPATHWAYS, 16 genes). Familial hyperlipidemia type 2 is familial hypercholesterolemia (FH), a common autosomal dominant inherited disorder defined by pathologically elevated plasma LDL-cholesterol from birth. The molecular defect lies in the hepatic LDL receptor (LDLR) pathway: loss-of-function variants in LDLR eliminate or reduce the receptor itself; APOB defects impair LDL binding to LDLR; PCSK9 gain-of-function mutations accelerate lysosomal degradation of LDLR; LDLRAP1 mutations disrupt clathrin-mediated internalization of the LDLR-LDL complex. Any of these defects impairs hepatic clearance of circulating LDL particles, causing LDL accumulation that drives premature atherosclerosis and coronary artery disease. This set is mechanistically distinct from WP_FAMILIAL_HYPERLIPIDEMIA_TYPE_1 (which represents the LPL/chylomicron/triglyceride axis); type 2 is defined by cholesterol and LDL-receptor biology.
