# MYGENESET:KEGG_ENDOMETRIAL_CANCER  KEGG_ENDOMETRIAL_CANCER

**MYGENESET:KEGG_ENDOMETRIAL_CANCER** — KEGG_ENDOMETRIAL_CANCER (msigdb, C2:CP:KEGG_LEGACY)

## Description

KEGG legacy endometrial cancer pathway gene set (MSigDB C2 curated gene set). This is a cancer disease pathway gene set, not a cell-type signature. The member genes are enriched for PI3K/AKT signaling components (PTEN, PIK3CA, AKT1), reflecting the predominant driver in Type I endometrioid endometrial carcinoma, as well as Wnt/ beta-catenin pathway members (CTNNB1, APC, GSK3B, AXIN), RAS/MAPK effectors (KRAS, BRAF, MAP2K1, MAPK1), cell adhesion regulators including E-cadherin (CDH1), and apoptosis modulators (TP53, CASP3, BCL2-family). PTEN loss is the most frequent somatic event in endometrial cancer (~60-80% of Type I tumours), constitutively activating PI3K-AKT survival signalling. The pathway also captures Wnt/beta-catenin dysregulation driving endometrioid carcinogenesis, MAPK proliferative signalling, E-cadherin loss enabling myometrial invasion, and TP53 alteration characteristic of the more aggressive Type II (serous) subtype.
