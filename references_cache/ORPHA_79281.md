# ORPHA:79281  Alpha-N-acetylgalactosaminidase deficiency type 3

**ORPHA:79281** — Alpha-N-acetylgalactosaminidase deficiency type 3 (Clinical subtype, Subtype of disorder)

## Synonyms

- NAGA deficiency type 3
- Schindler disease type 3

## Definition

A rare clinically heterogeneous type of NAGA deficiency with developmental, neurologic and psychiatric manifestations presenting at an intermediate age.

## Inheritance

- Autosomal recessive

## Natural history

- Age of onset: Childhood
