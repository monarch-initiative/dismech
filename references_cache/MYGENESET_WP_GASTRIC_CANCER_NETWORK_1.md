## Description

WikiPathways gastric cancer network 1 gene set (MSigDB C2:CP:WIKIPATHWAYS, 28 genes). This is a cancer disease pathway gene set representing the major mechanistic arms of gastric (stomach) adenocarcinoma: growth factor receptor signaling (EGFR, ERBB2/HER2) driving proliferation, Wnt/β-catenin pathway components (CDH1/E-cadherin, CTNNB1/β-catenin) coupling cell adhesion loss to oncogenic transcription, cell-cycle regulators (CCND1, CDK4/6), and apoptosis modulators (TP53, BCL2 family). The gene set captures the canonical oncogenic network activated in gastric carcinogenesis.
