# MYGENESET:HALLMARK_HYPOXIA  HALLMARK_HYPOXIA

**MYGENESET:HALLMARK_HYPOXIA** — HALLMARK_HYPOXIA (msigdb, H)

## Description

MSigDB Hallmark gene set of genes up-regulated in response to hypoxia (low oxygen tension). This is a condition/process signature, not a cell-type signature. Member genes are activated by the HIF-1 (hypoxia-inducible factor 1) transcriptional programme and reflect the canonical cellular adaptations to oxygen deficiency: a glycolytic metabolic shift (Warburg-like aerobic glycolysis), induction of angiogenesis via VEGF, and broader stress-response remodelling. The set is compiled from the Hallmark (H) collection of coherent, well-defined biological states.
