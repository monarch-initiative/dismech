# MYGENESET:HALLMARK_TNFA_SIGNALING_VIA_NFKB  HALLMARK_TNFA_SIGNALING_VIA_NFKB

**MYGENESET:HALLMARK_TNFA_SIGNALING_VIA_NFKB** — HALLMARK_TNFA_SIGNALING_VIA_NFKB (msigdb, H)

## Description

MSigDB Hallmark gene set of genes up-regulated by TNF-alpha signaling through NF-kB. This is a stimulus-response signature, not a cell-type signature: member genes are induced by TNF-alpha (tumor necrosis factor) through the canonical IKK–NF-kB axis and represent the early inflammatory transcriptional response. The set includes cytokines, chemokines, adhesion molecules, anti-apoptotic factors, and other NF-kB target genes up-regulated in diverse cell types upon TNF-alpha stimulation. It is widely used to infer NF-kB pathway activity and TNF-driven inflammation in bulk and single-cell RNA-seq data.
