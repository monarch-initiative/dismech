# A model for dengue disease with variable human population.
**Authors:** Esteva L, Vargas C

A model for the transmission of dengue fever with variable human population size 
is analyzed. We find three threshold parameters which govern the existence of 
the endemic proportion equilibrium, the increase of the human population size, 
and the behaviour of the total number of human infectives. We prove the global 
asymptotic stability of the equilibrium points using the theory of competitive 
systems, compound matrices, and the center manifold theorem.
