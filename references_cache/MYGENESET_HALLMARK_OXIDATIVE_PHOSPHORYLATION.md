# MYGENESET:HALLMARK_OXIDATIVE_PHOSPHORYLATION  HALLMARK_OXIDATIVE_PHOSPHORYLATION

**MYGENESET:HALLMARK_OXIDATIVE_PHOSPHORYLATION** — HALLMARK_OXIDATIVE_PHOSPHORYLATION (msigdb, H)

## Description

MSigDB Hallmark gene set of genes encoding proteins of the mitochondrial oxidative phosphorylation and electron transport chain (ETC) machinery. This is a metabolic-process signature, not a cell-type signature; the member genes are predominantly nuclear-encoded subunits of ETC complexes I–V (NADH dehydrogenase, succinate dehydrogenase, ubiquinol–cytochrome c reductase, cytochrome c oxidase, and ATP synthase), together with components of the TCA cycle and mitochondrial ribosomal proteins. The hallmark represents the core mitochondrial bioenergetic programme that is active in virtually all aerobically respiring human cell types.
