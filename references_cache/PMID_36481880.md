Retinitis pigmentosa (RP) is a set of symptoms including tunnel vision, night 
blindness, and progressive vision loss, stemming from a very heterogeneous set 
of causes-it can result from a several different kinds of mutations 
(non-syndromic) in conjunction with other symptoms, as part of a larger syndrome 
(syndromic), or secondary to an organ system disease state (secondary RP). This 
chapter explores and elucidates these various causes of RP.
