# MYGENESET:BENPORATH_PRC2_TARGETS  BENPORATH_PRC2_TARGETS

**MYGENESET:BENPORATH_PRC2_TARGETS** — BENPORATH_PRC2_TARGETS (msigdb, C2:CGP)

## Description

MSigDB C2:CGP gene set of Polycomb Repressive Complex 2 (PRC2) target genes in human embryonic stem cells. Member genes bear the H3K27me3 histone mark and are bound by the PRC2 subunits SUZ12 and EED, as identified by ChIP-seq in hES cells (Ben-Porath et al. 2008, PMID:18443585; 650 genes). The set is dominated by developmental transcription factors — HOX cluster genes (HOXA, HOXB, HOXC, HOXD), PAX, GATA, CDX, DLX, NKX, HAND, and TWIST family regulators — that are kept epigenetically silenced by PRC2 to maintain pluripotency in ES cells. This is a chromatin/genetic-perturbation target set (genes repressed by polycomb), not a directional expression signature; direction is NA.
