# MYGENESET:KEGG_PARKINSONS_DISEASE  KEGG_PARKINSONS_DISEASE

**MYGENESET:KEGG_PARKINSONS_DISEASE** — KEGG_PARKINSONS_DISEASE (msigdb, C2:CP:KEGG_LEGACY)

## Description

KEGG legacy Parkinson's disease pathway gene set (MSigDB C2 curated gene set). This is a disease pathway gene set, not a cell-type signature. The member genes are heavily enriched for mitochondrial electron-transport-chain subunits (oxidative phosphorylation), ubiquitin-proteasome system components (parkin/PINK1 pathway), dopaminergic neuron biology, and apoptosis regulators, reflecting the major mechanistic arms of the KEGG Parkinson's disease pathway representation. As a legacy/core pathway it captures established mechanism and omits more recently characterised arms such as ferroptosis (recorded below as a membership_gap rather than a false association).
