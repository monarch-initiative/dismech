# MYGENESET:KEGG_RENAL_CELL_CARCINOMA  KEGG_RENAL_CELL_CARCINOMA

**MYGENESET:KEGG_RENAL_CELL_CARCINOMA** — KEGG_RENAL_CELL_CARCINOMA (msigdb, C2:CP:KEGG_LEGACY)

## Description

KEGG legacy renal cell carcinoma pathway gene set (MSigDB C2 curated gene set). This is a cancer disease pathway gene set, not a cell-type signature. The member genes capture the dominant molecular mechanisms of renal cell carcinoma: VHL tumour suppressor loss leading to constitutive HIF stabilisation (pseudohypoxia), VEGF-driven angiogenesis, growth-factor receptor signalling via RAS/MAPK and PI3K/AKT pathways, and cell cycle dysregulation. Clear-cell RCC (ccRCC), the most prevalent subtype, is almost universally driven by VHL inactivation, making the HIF transcriptional programme the defining molecular axis of this pathway representation.
