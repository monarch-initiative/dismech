**MYGENESET:HALLMARK_INTERFERON_GAMMA_RESPONSE** — HALLMARK_INTERFERON_GAMMA_RESPONSE (msigdb, H)

## Description

MSigDB Hallmark gene set of genes up-regulated in response to interferon-gamma (IFN-γ). This is a stimulus-response signature, not a cell-type signature: member genes are induced in diverse cell types by IFN-γ signalling through the JAK1/JAK2–STAT1 axis and include MHC class I and II presentation machinery, interferon-stimulated genes (ISGs), antiviral effectors, and immune co-stimulatory molecules. The set defines the canonical IFN-γ transcriptional response and is widely used to infer IFN-γ pathway activity in bulk and single-cell RNA-seq data.
