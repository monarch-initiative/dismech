# MYGENESET:KEGG_PATHOGENIC_ESCHERICHIA_COLI_INFECTION  KEGG_PATHOGENIC_ESCHERICHIA_COLI_INFECTION

**MYGENESET:KEGG_PATHOGENIC_ESCHERICHIA_COLI_INFECTION** — KEGG_PATHOGENIC_ESCHERICHIA_COLI_INFECTION (msigdb, C2:CP:KEGG_LEGACY)

## Description

KEGG legacy Pathogenic Escherichia coli infection pathway gene set (MSigDB C2 curated gene set). This is a disease pathway gene set, not a cell-type signature. The member genes reflect the host-cell molecular response to enteropathogenic (EPEC) and enterohemorrhagic (EHEC) E. coli, Gram-negative enteric pathogens that colonise intestinal epithelial cells via attaching-and-effacing (A/E) lesions. These bacteria inject type III secretion system (T3SS) effectors directly into host cells to hijack the actin cytoskeleton, forming characteristic pedestal-shaped actin structures beneath attached bacteria, and to disrupt epithelial tight junctions, increasing intestinal permeability. Dominant gene classes include actin cytoskeleton regulators (CDC42, RHOA, RAC1, N-WASP/WASL, Arp2/3 complex subunits), tight junction proteins (claudins, occludin/OCLN, ZO proteins/TJP1-3), and Rho GTPase signalling components, together with downstream innate immune signalling genes.
