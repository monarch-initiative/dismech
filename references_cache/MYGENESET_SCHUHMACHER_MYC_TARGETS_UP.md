# MYGENESET:SCHUHMACHER_MYC_TARGETS_UP  SCHUHMACHER_MYC_TARGETS_UP

**MYGENESET:SCHUHMACHER_MYC_TARGETS_UP** — SCHUHMACHER_MYC_TARGETS_UP (msigdb, C2:CGP)

## Description

MSigDB C2:CGP signature of genes up-regulated upon conditional MYC induction in P493-6 Burkitt's lymphoma B cells (Schuhmacher 2001, PMID:11139609). The P493-6 cell line carries a tetracycline-repressible MYC transgene, allowing clean induction of MYC expression; 79 genes were identified by oligonucleotide microarray as transcriptionally activated downstream of MYC. This is an independent MYC genetic-perturbation UP signature, distinct from the DANG MYC targets set (different cell model and experimental system), and defines the canonical transcriptional program driven by MYC induction: ribosome biogenesis, translation, nucleotide biosynthesis, and cell growth.
