# MYGENESET:KEGG_TYPE_II_DIABETES_MELLITUS  KEGG_TYPE_II_DIABETES_MELLITUS

**MYGENESET:KEGG_TYPE_II_DIABETES_MELLITUS** — KEGG_TYPE_II_DIABETES_MELLITUS (msigdb, C2:CP:KEGG_LEGACY)

## Description

KEGG legacy type 2 diabetes mellitus pathway gene set (MSigDB C2 curated gene set). This is a disease pathway gene set representing the pathophysiology of type 2 diabetes, characterized by insulin resistance in peripheral tissues and progressive pancreatic beta-cell failure. Member genes are enriched for insulin receptor signaling components (INSR, IRS1, IRS2, PIK3CA, AKT1/2), downstream MAPK and PI3K-AKT cascade mediators, glucose and fatty acid metabolic enzymes, and pancreatic beta-cell secretory machinery. The gene set reflects the two cardinal defects of T2D: impaired insulin signal transduction (insulin resistance) and insufficient insulin secretion, along with downstream metabolic dysregulation in liver, skeletal muscle, and adipose tissue.
