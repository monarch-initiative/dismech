# MYGENESET:KEGG_ALZHEIMERS_DISEASE  KEGG_ALZHEIMERS_DISEASE

**MYGENESET:KEGG_ALZHEIMERS_DISEASE** — KEGG_ALZHEIMERS_DISEASE (msigdb, C2:CP:KEGG_LEGACY)

## Description

KEGG legacy Alzheimer's disease pathway gene set (MSigDB C2 curated gene set). This is a disease pathway gene set, not a cell-type signature. The member genes are heavily enriched for mitochondrial electron-transport-chain and ATP-synthase subunits (oxidative phosphorylation), amyloid precursor protein processing machinery, apoptosis regulators, and calcium-signaling components, reflecting the major mechanistic arms of the KEGG Alzheimer's disease pathway representation.
