**MYGENESET:HALLMARK_EPITHELIAL_MESENCHYMAL_TRANSITION** — HALLMARK_EPITHELIAL_MESENCHYMAL_TRANSITION (msigdb, H)

## Description

MSigDB Hallmark gene set defining epithelial-to-mesenchymal transition (EMT). This is a process/phenotype signature, not a cell-type signature. Member genes are enriched for extracellular matrix components (collagens, fibronectin, laminins), ECM remodelling enzymes (MMPs, LOX), cell adhesion molecules (integrins, cadherins), and TGF-beta pathway components, collectively capturing the transcriptional reprogramming that drives epithelial cells to lose apical-basolateral polarity, dissolve intercellular junctions, degrade basement membrane, and acquire migratory mesenchymal properties.
